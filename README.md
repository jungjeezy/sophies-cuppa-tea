# Sophie's Cuppa Tea POC

This folder contains a standalone local proof of concept for a fresh Sophie's Cuppa Tea homepage direction.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Run locally
From this folder, run:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Intent
This is not a production build. It is a shareable visual and strategic concept designed to:
- reframe the brand around authority, ritual, and guided discovery
- show a cleaner homepage direction
- keep room for real content, products, and assets to be filled in later
